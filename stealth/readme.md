# Installation

This set of little helpers for WordPress hacking can either be installed as a plugin or just dumped into the webroot.

## Plugin Installation Methods

```
wp plugin install https://
```

## Web Installation Methods

```
cd /var/www/html
wget https://
unzip *.zip
```
